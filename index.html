About

무역아카데미에서 진행하는 SMART Cloud IT 마스터 과정에서 프로그래밍에 대한 공부했습니다. 이후 일본 IT 기업에 개발 엔지니어로 취업하셨으나 네트워크 팀에 배속되어 2019년 10월부터 2021년 10월까지 2년간 네트워크 엔지니어로 근무해왔으며, 제로베이스에서 공부와 실무경험을 통해 6개월 만에 CCNA를 합격하였습니다. 그러나 한국에서 배운 프로그래밍에 대한 즐거움이 그리워 다시 한번 프로그래머로서 도전하기 위해 준비 중입니다. 현재 기존에 배운 내용을 복습하고 있으며, 복습 이후 2개월 만에 Oracle Java Silver를 합격하였습니다. 이러한 좌절하지 않는 성격과 학습에 대한 의욕과 열정을 살려서 입사 후 업무에 바로 적응할 수 있도록 노력하겠습니다.
